## 核心 UI 語言

- **焦慮分數環** `AnxietyScoreRing`:conic 進度環,門檻 40 / 70 變色(綠→淡橘→橘),數字 monospacedDigit,動畫 spring 0.8s
- **情緒三級** `EmotionBadge`:安心(綠)/ 有點緊張(橘)/ 波動較大(紅)
- **抽卡** `DrawCard`:5 種卡型 — 冷靜觀察 / 大盤影響 / 小心震盪 / 個股事件 / 信心恢復
- **新手翻譯** `ExplanationBlock`:每個數據頁都配一段白話解釋
- **免責** `DisclaimerBlock`:所有解釋頁尾「內容僅供資訊參考,不構成投資建議」
- **語氣**:安撫優先、不催促決策 —「先看懂原因,不用急著做決定」

## 設計原則(承自需求文件)

1. 情緒安撫優先於資訊密度 — 一頁只講一件事
2. 白話翻譯,不用術語 — 看不懂的數字都配一句人話
3. 柔和色彩,避免刺激 — 不用高飽和紅綠漲跌色塊
4. 原生 iOS 慣例 — SF Pro / PingFang TC、原生 Toggle、Sheet、Tab Bar
5. 不是投資建議 — 全站不出現買/賣/加碼字眼

## 畫布新增段落(第十三輪)· 每日抽卡包 + AI 信任系統 ★最新

「每日抽籤」升級為「每日抽卡包」,並內建 AI 信任系統(可驗證/可溯源/可解釋)。資料基準 2025/12/31,示範標的台積電 2330。開包流程以深色模式展示光效。

- `15a` 今日卡包入口 — 卡包封面 + 「為什麼是這檔」+ 出處 chip
- `15b–15e` 開包動畫 4 關鍵幀 — 撕開光效 → 事實卡飛出翻面 → 推論卡 → 卡疊完成態(含社群溫度計 widget);全程右上「跳過」、tap 加速
- `15f` 事實卡閃卡版 — conic 流光邊框 + 掃光,觸發「股價創歷史新高」(寫死數據事件)
- `15g` 推論卡推理鏈展開 — 3 步皆數字、每步附出處 chip,虛線術語可開名詞小卡
- `15h` 陪伴卡 — 暖色 LXGW WenKai 手寫感,零買賣暗示
- `15i` 出處 chip bottom sheet — 欄位/原始值/計算方式/資料日期/來源 + 「券商 App 也查得到」
- `15j` 卡包架收藏頁 — 橫滑聚焦、產業色封面、新洞察發光、歷史卡片圖鑑
- `15k` 週末體檢回顧 — 「AI 本週誠實度」對帳(2/3 應驗照實呈現)+ 本週特別卡包

三張卡視覺基因:事實卡灰底冷靜等寬數字 / 推論卡藍(#5E6FD8)標示「AI 推論」 / 陪伴卡暖色手寫。每頁 footer:「本內容為現況描述與風險提示,非投資建議｜資料截至 2025/12/31 · 示範標的籃子」。

## 畫布新增段落(第十輪)· 每日抽籤(御神籤)

每日抽卡改為日式搖籤儀式。籤詩固定三欄位:**持股與狀態**(每檔對應六級籤等)/ **說明**(可能發生的事 + 適合進場嗎)/ **注意事項**(最近事件、需留意的點)。

- `12a` 搖籤入口 — 籤筒待機微晃(rock ±2.5° 3.2s),CTA「開始搖籤」,每天一支、收盤後更新
- `12b` 搖籤動畫 — 筒身傾斜 26° 高頻搖動(±13° 0.55s),籤支「十四」從筒口彈出(stickPop 2s)
- `12c` 籤詩結果 — 直式籤詩紙(紅雙框)+ 籤等大字 + 六級對照列 + 三欄位 + 免責
- `12d` 籤等對照 — 六級顏色 tokens(紅偏吉 / 綠偏凶,呼應台股漲跌色)+ 文案語氣;凶籤走安撫視覺

### 影像匯入 · 動畫規格

- 7a→7b:截圖框 scale 0.94→1 + fade,spring 0.5s
- 掃描線:translateY 6% → 86% 往返,easeInOut 3s 無限循環
- spinner:rotate 360°,linear 0.9s
- 步驟完成:圈染綠 + 打勾 path draw 0.3s,light haptic
- 7b→7c:三步全完成後停 0.4s 再 push
- 持股列進場:fade + 上移 12pt,stagger 60ms/列
- 低信心提醒:列進場後延遲 0.6s 淡入金色描邊(安撫原則:不搖晃、不用紅色)
- 修改股數:點列 → 原生數字鍵盤,列展開 spring 0.4s
- CTA:按下 scale 0.97;成功打勾 + medium haptic 後返回持股頁
- 辨識失敗:不出紅色錯誤畫面,文案「沒認出來,換張清楚一點的截圖試試」+ 重選 / 改手動輸入
