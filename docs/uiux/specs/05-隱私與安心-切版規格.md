# 05 · 隱私與安心 切版規格(10a / 10b / 10c / 10d)

單位 pt。基準 iPhone 16 Pro 402 × 874。頁面內距:上 88 / 左右 24 / 下 40。
色彩/字級 tokens 沿用 `00-通用規格`。

## 核心原則

1. **具體白話,不用空話**:不寫「銀行級加密」,寫「截圖辨識完即刪除」「登出即清空」。
2. **只收最小資料是賣點**:slogan「我們連你的券商帳號都沒有」。只存 代號+股數+均價,不碰券商帳密、不做 API 連結、不儲存 Email。
3. **刪除要真的即時、可見**:一鍵刪除後逐類顯示刪了幾筆,不是「已收到您的請求」。
4. **關鍵時刻就地說明**:在動作發生的當下給一句話,不藏在條款裡。

## 10b · 你的資料存在哪裡 `DataMapView`

入口:10a 儀表板列尾「你的資料存在哪裡?」/ 設定頁第二列。圖解式,三個區塊由上而下:

- **區塊卡**(白底 R20,內距 18/20,卡間 12),每卡:位置 icon 圈 42 + 位置名 15 Heavy + 描述列 ×N(色點 7 + 字 12 #5C5850 行高 1.7)
  1. **這台手機上**(icon `iphone`,圈 bg #EEEEFA):登入金鑰(Keychain 加密)/ 顯示偏好(金額模糊、Face ID 開關)
  2. **我們的伺服器**(icon `server.rack`,圈 bg #FBEFDF):一組匿名編號 / 持股三欄(代號・股數・均價)/ 異動與抽卡紀錄 — 註「用 HTTPS 傳輸;登入 Email 不儲存,只用來換匿名編號」
  3. **永遠不會有**(icon `hand.raised`,圈 bg #EAF2EC):券商帳密 / 下單權限 / 身分證件 — 底色改 #FAF8F3
- **流程註解盒**(bg #F1F1FB R14,內距 12/14,距 16):「對帳單截圖只用來辨識代號與股數:優先在你的手機上辨識(不上傳);需要上傳時走加密連線、辨識完即刪除、不寫入紀錄檔。」12 #4A4770 行高 1.7

## 10c · 敏感畫面防護

三件事,設定頁「隱私與安心」section 提供開關,元件散落各頁:

### 金額模糊 `AmountBlurToggle`
- 持股頁 nav bar 右側眼睛鈕(`eye` / `eye.slash`,20pt #8A857C,點擊範圍 44)
- 開啟時:所有金額數字(現價、損益、均價、市值)以 `redacted + blur 6` 顯示;股票名稱與漲跌 pill 不遮
- 狀態全 App 共用(持股列表 / 個股詳情 / 券商分帳 / 今日總覽同步),存本機偏好;設定頁可設「預設模糊」
- 切換動畫:blur 0.2s easeOut + light haptic

### 背景遮罩 `SnapshotShield`
- App 進入 App Switcher / 背景時,全螢幕蓋米杏底 + App icon + 「內容已遮罩」13 #8A857C
- 回前景 fade out 0.2s;永遠開啟,不提供關閉(成本零、風險高)

### Face ID 鎖 `HoldingsLock`
- 設定頁 toggle「進入持股頁需要 Face ID」,預設關
- 開啟後:切到持股 tab → 全頁遮罩(App icon + 「用 Face ID 解鎖持股」+ 解鎖鈕)→ LocalAuthentication
- 解鎖狀態維持到 App 回背景;失敗可重試、fallback 裝置密碼
- 無生物辨識的裝置:toggle 顯示不可用說明

## 10d · 關鍵時刻就地說明 `TrustNote`

共用元件:`lock.shield` icon 13pt #6E9A7F + 字 11 #8A857C 行高 1.6,無底色、無邊框(輕,不打斷)。

| 位置 | 文案 |
|---|---|
| 掃描入口(7a,匯入按鈕下方) | 只辨識代號與股數,辨識完即刪除;不取得帳號或下單權限 |
| 首次輸入持股(3a,確認鈕上方) | 只需要代號和股數,均價可以之後再補;不會要你連券商帳號 |
| 庫存分析頁頂(8a,大標下方) | 分析只用你的持股組合(代號+權重),不含任何身分資料 |
| 隱私儀表板入口列(設定頁) | 我們有什麼、沒有什麼,3 秒看完 |

## 技術對應(工程實作)

| 項目 | 實作 |
|---|---|
| 截圖 OCR | 優先裝置端 Vision framework(`VNRecognizeTextRequest`);辨識不到才上傳 GPT,TLS + 即用即刪、不落地 log |
| 一鍵刪除 | `DELETE /privacy/all`:當下刪除 持股/異動/抽卡/成就/提醒 五類,回傳各類刪除筆數供 UI 逐項顯示 |
| 資料摘要 | `GET /privacy/summary`:回傳各類現存筆數,儀表板即時顯示 |
| 匿名化 | 登入僅發匿名編號 session;Email 不入庫(後端無用戶資料表) |
| 本機偏好 | 金額模糊 / Face ID 開關存 UserDefaults(per-user scoped);登入金鑰存 Keychain |
| 隱私標籤 | App Privacy Nutrition Label:僅「與你連結的資料:財務資訊(持股)」;不含聯絡資訊/識別碼追蹤 |
