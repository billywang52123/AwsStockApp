# 03 · 動畫規格(庫存分析 + 個股 AI 觀點)

原則:安撫優先 — 不搖晃、不閃爍、不用高飽和紅綠;所有動畫可被「減少動態效果」設定關閉。

## 8a 庫存分析總覽

- 頁面進場:卡片由上而下 fade + 上移 12pt,stagger 80ms
- 總市值 / 損益數字:count-up 0.8s easeOut,`monospacedDigit` 防跳動
- 分數數字(78 / 72):count-up 0.6s,延遲 0.2s
- **曝險 bar**:分段由左至右依序長出(scaleX,anchor leading),每段 0.4s easeOut、stagger 80ms;圖例隨對應分段 fade-in
- badge「科技類達 68.9%」:bar 全部長完後 fade-in 0.3s

## 8b 持股明細

- 列進場:fade + 上移 12pt,stagger 60ms/列
- 權重 bar:列進場完成後填色 0→權重%,0.6s easeOut
- 損益數字:不做逐字動畫,直接顯示(避免盯著賺賠跳動)
- 點列 → 進入個股觀點詳情(8e):原生 push

## 8c 風險提醒

- 卡片進場:fade + 上移 12pt,stagger 120ms;描邊色延遲 0.4s 淡入(先白話、後提醒)
- badge(優先檢查 / 注意):卡片定位後 scale 0.8→1 spring 0.4s,**不重複跳動**
- CTA 按下:scale 0.97;釋放 spring 回彈 + light haptic

## 8d 個股 AI 觀點總覽

- 統計 chips:由左至右 fade + scale 0.9→1,stagger 60ms
- 列進場:同 8b(fade + 上移 12pt,stagger 60ms)
- OutlookBadge:列定位後 fade-in 0.25s,無彈跳

## 8e 個股觀點詳情

- push 進場後:綜合觀點卡 fade + 上移 → 訊號卡 stagger 90ms → 白話總結最後
- **溫度計滑標**:自 50%(中性)spring 0.8s 滑至目標分數位置,單次定位不回彈多次;伴隨 light haptic
- 溫度計軌道漸層:靜態,不流動
- 訊號卡方向標籤(→ 短線偏空 等):卡片定位後 fade-in 0.3s

## Haptics 總表

- light:CTA 按下、溫度計定位、chips 選取
- medium:成功動作完成(沿用既有規格)
- 不使用 error/warning haptic(安撫原則)

## 每日抽卡包 開包動畫(15b–15e,詳見 06 規格)

- KF1 撕開:光帶 `glowPulse` 1.6s 循環、背景光暈 `packBloom` 3.2s 循環,總時長 ~600ms 後自動進 KF2
- KF2 事實卡飛出:卡體從撕開處 translateY + rotate(-6°)彈出,spring 0.5s,下方投影同步淡入
- KF3 推論卡接續:事實卡縮小 scale 0.96 貼頂;推論卡 translateY 上移 + rotate(4°)定位,spring 0.45s;**點一下或 1.5s 自動**進下一關鍵幀
- KF4 卡疊覆蓋態:三卡背面朝上收攏成疊(前景 1 張 + 後方 2 張偏移 ±44pt/rotate ∓6°/scale 0.92),收攏 spring 0.4s;前景卡待機 `flipHint` rotateY 擺動(3.6s 循環,-24° 擺幅)提示可翻
- 滑動選卡(15e):swipe 觸發卡疊輪轉——前景卡退到後方、下一張升為前景,位置/rotate/scale/opacity 全程插值,spring 0.45s,light haptic
- 點擊翻牌:前景卡 rotateY 0→180°(perspective 1100pt),0.5s easeInOut,50% 處切換卡背/卡面內容;翻至正面即完成態,medium haptic
- **閃卡翻牌爆光**:翻開的卡是閃卡時——卡背常駐金色光暈呼吸(`flashBurst` 2.2s,radial 金橘 blur 10)+「✦ 內有閃卡」pill;翻至 90° 時光暈 scale 1→1.6 爆開並拉高亮度(0.25s easeOut),落定正面後接閃卡常駐效果(conic 流光邊框 5s + 掃光 3.6s);爆開瞬間 medium haptic(僅一次)。非閃卡翻牌無光效
- 卡背待機微光:徽記呼吸光暈 `emblemPulse` 2.6s、光芒層 `rayRotate` 14s linear(僅前景卡跑動畫,後方卡靜止省電)

## 分享卡片(15L/15m)

- 「⤴ 分享這張卡」按下 → bottom sheet spring 由下滑入 0.35s + 背景 blur/scrim fade 0.2s,light haptic
- 卡樣式預覽的迷你卡:掃光 `holoSweep` 4s 循環,閃卡另帶 pill 發光
- 選定去向 → 卡圖(1080×1350)於背景合成後叫起系統 share sheet;「存成圖片」寫入相簿後 medium haptic + 短暫「已儲存」toast
- 貼文預覽(15m)卡圖:旋轉光芒 `rayRotate` 18s、頂部極光暈靜態、掃光 4.4s、徽記 `emblemPulse`——靜態存檔時取單一幀(去背光效轉為固定漸層,避免匯出成動圖)
- 全程:tap 任一處加速跳到下一關鍵幀;右上「跳過」直達 15e 完成態,不觸發任何一段動畫
- 完成態切換(15f↔15g↔15h):swipe 觸發 translateX 過渡 0.3s easeOut,light haptic;滑向未翻開的卡先播翻牌動畫再顯示內容
- 閃卡(15f):conic 邊框 5s linear 無限旋轉、掃光 3.6s ease-in-out 循環,不可暫停(閃卡狀態為持續態,非一次性動畫)
- 出處 chip → sheet(15i):背景 blur 2px + scrim fade 0.2s;sheet spring 由下滑入 0.35s
- 卡包架(15j)新洞察光暈:`newGlow` 呼吸 2.8s 循環;橫滑切換卡片:snap scroll,相鄰卡 scale/opacity 隨滑動位置插值
- 週末體檢(15k)banner 全息掃光:4.5s ease-in-out 循環;誠實度分段進度條首次出現時由左至右長出,0.6s easeOut

Haptics 補充:開包每次關鍵幀切換 light haptic;閃卡出現時 medium haptic(僅一次,不重複)。
