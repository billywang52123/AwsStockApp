## 畫布結構(由上而下)

### 2a · iOS 切版規格卡(開發交付)
單位 pt,基準機型 iPhone 16 Pro(402 × 874 pt),含 5 張規格卡:
1. **佈局網格與安全區** — 左右邊距 24pt、卡片間距 12pt、Tab Bar 50pt + Home Indicator 24pt
2. **元件尺寸** — 按鈕 54/50pt、進度環 196pt、抽卡 240×364pt、最小點擊 44×44pt
3. **字級** — LargeTitle 30–34pt Heavy → Caption 11–12pt,全部掛 Dynamic Type,中文行高 ≥1.6
4. **色彩 Tokens** — 方向 A / B 對照表(Asset Catalog 命名)
5. **Redline 首頁** — 首頁逐項間距標註 + 進度環動畫規格

### 1a · 方向 A「暖陽米杏」
更溫暖、有儀式感:米白底 `#F7F4EE` + 藍紫主色 `#7B7FD4` + Noto Serif TC 儀式標題 + 暖色柔陰影,圓角 18–28。

### 1b · 方向 B「沉靜霧藍」
更冷靜、留白多:霧灰底 `#F3F4F7` + 靛藍主色 `#6E79C9` + 全 sans + 髮絲線分隔,圓角 12–18,陰影極輕。

兩方向各含 **13 個畫面 + 1 頁 Design System**。

## 核心 UI 語言

- **焦慮分數環** `AnxietyScoreRing`:conic 進度環,門檻 40 / 70 變色(綠→淡橘→橘),數字 monospacedDigit,動畫 spring 0.8s
- **情緒三級** `EmotionBadge`:安心(綠)/ 有點緊張(橘)/ 波動較大(紅)
- **抽卡** `DrawCard`:5 種卡型 — 冷靜觀察 / 大盤影響 / 小心震盪 / 個股事件 / 信心恢復
- **新手翻譯** `ExplanationBlock`:每個數據頁都配一段白話解釋
- **免責** `DisclaimerBlock`:所有解釋頁尾「內容僅供資訊參考,不構成投資建議」
- **語氣**:安撫優先、不催促決策 —「先看懂原因,不用急著做決定」

## 設計原則(承自需求文件)

1. 情緒安撫優先於資訊密度 — 一頁只講一件事
2. 白話翻譯,不用術語 — 看不懂的數字都配一句人話
3. 柔和色彩,避免刺激 — 不用高飽和紅綠漲跌色塊
4. 原生 iOS 慣例 — SF Pro / PingFang TC、原生 Toggle、Sheet、Tab Bar
5. 不是投資建議 — 全站不出現買/賣/加碼字眼
