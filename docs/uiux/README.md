## 畫布結構(由上而下)

### 2a · iOS 切版規格卡(開發交付)
單位 pt,基準機型 iPhone 16 Pro(402 × 874 pt),含 5 張規格卡:
1. **佈局網格與安全區** — 左右邊距 24pt、卡片間距 12pt、Tab Bar 50pt + Home Indicator 24pt
2. **元件尺寸** — 按鈕 54/50pt、進度環 196pt、抽卡 240×364pt、最小點擊 44×44pt
3. **字級** — LargeTitle 30–34pt Heavy → Caption 11–12pt,全部掛 Dynamic Type,中文行高 ≥1.6
4. **色彩 Tokens** — 方向 A / B 對照表(Asset Catalog 命名)
5. **Redline 首頁** — 首頁逐項間距標註 + 進度環動畫規格

### 1a · 方向 A「暖陽米杏」
更溫暖、有儀式感:米白底 `#F7F4EE` + 藍紫主色 `#7B7FD4` + Noto Serif TC 儀式標題 + 暖色柔陰影,圓角 18–28。

### 1b · 方向 B「沉靜霧藍」
更冷靜、留白多:霧灰底 `#F3F4F7` + 靛藍主色 `#6E79C9` + 全 sans + 髮絲線分隔,圓角 12–18,陰影極輕。

兩方向各含 **13 個畫面 + 1 頁 Design System**。

## 核心 UI 語言

- **焦慮分數環** `AnxietyScoreRing`:conic 進度環,門檻 40 / 70 變色(綠→淡橘→橘),數字 monospacedDigit,動畫 spring 0.8s
- **情緒三級** `EmotionBadge`:安心(綠)/ 有點緊張(橘)/ 波動較大(紅)
- **抽卡** `DrawCard`:5 種卡型 — 冷靜觀察 / 大盤影響 / 小心震盪 / 個股事件 / 信心恢復
- **新手翻譯** `ExplanationBlock`:每個數據頁都配一段白話解釋
- **免責** `DisclaimerBlock`:所有解釋頁尾「內容僅供資訊參考,不構成投資建議」
- **語氣**:安撫優先、不催促決策 —「先看懂原因,不用急著做決定」

## 設計原則(承自需求文件)

1. 情緒安撫優先於資訊密度 — 一頁只講一件事
2. 白話翻譯,不用術語 — 看不懂的數字都配一句人話
3. 柔和色彩,避免刺激 — 不用高飽和紅綠漲跌色塊
4. 原生 iOS 慣例 — SF Pro / PingFang TC、原生 Toggle、Sheet、Tab Bar
5. 不是投資建議 — 全站不出現買/賣/加碼字眼

## 畫布新增段落(第五輪)· 持股影像匯入

- `7a` 截圖匯入入口 — 新增持股頁:券商庫存截圖匯入(相簿/拍照)優先、手動輸入為輔,附隱私說明(截圖辨識後即刪除)
- `7b` 辨識中 — 截圖掃描線動畫 + 三步驟進度(讀取畫面 → 辨識代號 → 對應股名股數)
- `7c` 結果確認 — 辨識出的持股列表,股數可直接修改;低信心列以金色卡提醒「幫我確認一下」
- `7d` 切版與動畫規格 — 開發交付卡(見下方詳細)

### 影像匯入 · 切版規格(單位 pt)

- **7a 匯入卡**:R24,內距 24/22,陰影 y12 blur28 rgba(120,100,70,.10);「最快」badge #E4B384 pill 11pt Bold,凸出卡頂 -11
- **icon 方塊**:匯入卡 54 R17(#EEEEFA);手動列 46 R15(#F1EDE3)
- **雙按鈕**(從相簿選截圖 / 直接拍照):高 50 R16,等寬,間距 10;主按鈕 #7B7FD4 陰影 y8 blur20 30%,次按鈕白底 1.5pt 描邊 #E3DFD4
- **隱私盒**:#F1F1FB R16,內距 14/16,字 12pt #4A4770
- **7b 截圖預覽框**:210 × 380 R22,陰影 y16 blur36 16%;placeholder 45° 斜紋 10pt
- **掃描線**:高 3,紫色漸層(兩端透明),光暈 blur18 60%
- **步驟圈**:24,描邊 2.5;完成 #6E9A7F 實心打勾 / 進行中 spinner / 待處理描邊 #E3DFD4;列間距 14
- **7c 持股列**:R18,內距 16/18,頭像圈 44;股數 16pt Bold monospacedDigit;鉛筆 icon 14pt,整列可點(≥44×44)
- **低信心列**:底 #FDF6E9 + 1.5pt 描邊 #ECD9B8;提示字 12pt #B0813F
- **CTA「加入這 N 檔持股」**:高 54 R18,陰影 y10 blur26 35%

### 影像匯入 · 動畫規格

- 7a→7b:截圖框 scale 0.94→1 + fade,spring 0.5s
- 掃描線:translateY 6% → 86% 往返,easeInOut 3s 無限循環
- spinner:rotate 360°,linear 0.9s
- 步驟完成:圈染綠 + 打勾 path draw 0.3s,light haptic
- 7b→7c:三步全完成後停 0.4s 再 push
- 持股列進場:fade + 上移 12pt,stagger 60ms/列
- 低信心提醒:列進場後延遲 0.6s 淡入金色描邊(安撫原則:不搖晃、不用紅色)
- 修改股數:點列 → 原生數字鍵盤,列展開 spring 0.4s
- CTA:按下 scale 0.97;成功打勾 + medium haptic 後返回持股頁
- 辨識失敗:不出紅色錯誤畫面,文案「沒認出來,換張清楚一點的截圖試試」+ 重選 / 改手動輸入
